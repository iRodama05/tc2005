# tc2005b-fj
## Descripción
Este es el repositorio personal en el que se irán subiendo las prácticas y laboratorios realizados dentro y fuera de clase. Como podrán observar, existen distintas carpetas en el repositorio (Labs, Practs, .github, Docs). Cada una de estas contendrá los trabajos que coincidan con su tipo. Sé que es bastante redundante, pero mencionaré en lista qué contiene cada carpeta con el fin de demostrar conocimientos en formato de Markdown.
- `Labs/`: Contiene los Laboratorios
- `Practs/`: Contiene las prácticas
- `Docs/`: Contiene documentos sueltos como `.pdf` y `.docx`.
- `.github/`: Contiene recursos útiles como la plantilla de Pull Request

### Aclaración sobre fechas no coincidentes:
Varias fechas de los commits no cuadrarán con la semana marcada en la malla debido a que originalmente estaba llenando un repositorio incorrecto y mudé todos los trabajos anteriores a este nuevo repositorio que será el utilizado de ahora en adelante.
